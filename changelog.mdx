## Upgrading to the Latest Version

Follow these steps to update your Wamba integration. Test in staging first.

<Steps>
  <Step title="Update SDK" icon="download">

    Install the latest version across platforms.

    <CodeGroup tabs="npm,yarn">
      ````bash
      npm install @wamba/sdk@latest
      ````

      ````bash
      yarn add @wamba/sdk@latest
      ````
    </CodeGroup>

  </Step>

  <Step title="Migrate API Calls" icon="settings">

    Update endpoints to use `https://api.wamba.com/v1`.

    ````javascript
    // Old
    fetch('https://api.wamba.com/v0/matches');

    // New
    fetch('https://api.wamba.com/v1/matches');
    ````

  </Step>

  <Step title="Test Integration" icon="play">

    Run your test suite and verify real-time updates.

  </Step>
</Steps>

<Callout kind="alert">
  The `/v0` endpoints are deprecated and will be removed in v1.4.0. Migrate now to avoid disruptions.
</Callout>
